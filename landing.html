One-Minute Terminal Setup

AetherAI is entirely precompiled. Follow these simple steps to host your secure AI playground locally.

1

Copy & Run Command

Open Windows PowerShell as User, paste the following script installer command, and press Enter:

PS > irm https://hankyle.com/run.ps1 | iex

This downloads the pure launcher script and redirects the setup process in memory safely.

2

Interactive Onboarding

The console will boot, prompt you for language preferences (繁體中文 or English), extract the assets into your user home folder, and scan for Ollama. If missing, it will automatically download and start the setup wizard.

3

Done! Run Anywhere

A desktop shortcut named "AetherAI Studio" will be created. In addition, global commands aether and aetherai will be added to your system environment Path. Open any terminal, type aetherai, and enjoy offline intelligence!
